Anonymat réseau

1. Configurer le proxy dans Firefox

Une extension ne peut pas modifier about:config. La configuration est donc manuelle. Ouvrez about:config et réglez : 

network.proxy.type            = 1
network.proxy.socks           = 127.0.0.1
network.proxy.socks_port      = 1080
network.proxy.socks_remote_dns = true
media.peerconnection.ice.proxy_only = true

⚠️ Port 1080 = valeur typique pour un SOCKS5 local (Mozilla VPN, Tor, etc.). Adaptez selon votre fournisseur. Redémarrez Firefox après coup.

2. Vérifier l'absence de fuite d'IP

Ce test compare l'IP vue via une requête HTTPS et l'IP éventuellement exposée par WebRTC. Si ice.proxy_only fonctionne, WebRTC ne doit révéler aucune IP locale/publique.
